Бизнес-процесс «Спорт-менеджмент»
Спорт-менеджмент – веб-сервис, помогающий менеджеру наблюдать за игроками в командных видах спорта, добавлять их, оформлять трансферы и отслеживать статистику и составы команд.

Основные сущности:
    • Пользователь (менеджер)
    • Роль
    • Игрок
    • Команда
    • Лига
    • Страна
    • Трансфер
Действия:
    • Регистрация нового пользователя
    • Подтверждение регистрации нового пользователя
    • Авторизация
    • Назначение роли пользователю (действие требует роли «администратор»)
    • Отзыв роли у пользователя (действие требует роли «администратор»)
    • Добавление команды (доступно администраторам)
    • Редактирование команды (доступно администраторам)
    • Добавление игрока (доступно администраторам)
    • Редактирование игрока (доступно администраторам)
    • Изменение прав доступа к посту (чтение для всех, чтение только для зарегистрированных, приватный пост, голосование, комментирование; доступно владельцу поста). В случае, если пост помечен как «для взрослых», нельзя выставить право «чтение для всех».
    • Добавление игрока (доступно любому пользователю при условии, что открыт доступ на комментирование к посту)
    • Редактирование игрока (доступно менеджеру игрока при условии, что нет ответов)
    • Удаление комментария к посту (действие доступно владельцу поста, автору комментария и пользователю с ролью «модератор»). Если к комментарию есть ответы, то вместо удаления нужно заменить текст комментария на текст «deleted».
    • Добавление оценки к посту (доступно любому пользователю при условии, что открыт доступ для голосования); ограничение – один пользователь может голосовать только один раз за пост
    • Изменение оценки к посту (доступно автору оценки)
    • Добавление оценки к комментарию (доступно любому пользователю, ограничение – один пользователь может голосовать один раз за комментарий)
    • Изменение оценки к комментарию (доступно автору оценки)
    • Удаление поста (действие доступно либо владельцу поста, либо пользователю с ролью «модератор»)
    • Просмотр поста в соответствии с правами доступа. В случае, если пост помечен как «для взрослых», проверяется возраст зарегистрированного пользователя. Незарегистрированным пользователям пост для взрослых не отображается.
    • Постраничный список постов пользователя. Список выдается с учетом прав доступа
    • Календарь постов пользователя (помесячный). Для каждого дня в календаре количество постов формируется с учетом прав доступа пользователя
    • Список пользователей с сортировкой по суммарному рейтингу
    • Список пользователей с сортировкой по популярности (учитывается кол-во просмотров + количество комментариев)
    • Список пользователей с сортировкой по недавнему обновлению (новому посту)
    • Top-100 постов по рейтингу
    • Top-100 постов по популярности
    • Top-100 недавно добавленных постов
    • Редактирование личных данных
    • Смена пароля
    • Подтверждение смены пароля
    • Запрос списка комментариев к посту
    • Подписка на автора
    • Удаление автора из подписки
    • Просмотр постов из подписки (с учетом прав доступа)
    • Добавление поста в избранное
    • Удаление поста из избранного
    • Просмотр постов в избранном
    • Список тэгов по алфавиту
    • Список тэгов по популярности
    • Поиск постов по тэгу
    • Список постов в разделе, постраничная выдача, сортировка от новых к старым
    • Список разделов с количеством постов в каждом
    • Журналирование действий
        ◦ Регистрация
        ◦ Подтверждение регистрации
        ◦ Запрос смены пароля
        ◦ Подтверждение запроса смены пароля
        ◦ Авторизация (успех, неуспех)
        ◦ Назначение роли
        ◦ Отзыв роли
        ◦ Удаление комментария
        ◦ Удаление поста
        ◦ Редактирование поста
        ◦ Редактирование комментария
